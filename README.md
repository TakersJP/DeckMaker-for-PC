# DeckMaker-for-PC
This Google Chrome extension will extend the size of [DeckMaker's](https://deck-maker.com) web page to fit your PC's screen.

こちらの Google Chrome拡張機能は、PCの画面に合わせて[デッキメーカー](https://deck-maker.com)のウェブサイトのサイズを拡張します。
